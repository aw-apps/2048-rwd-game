# 2048 網頁遊戲 (RWD 版本)

一個響應式設計的 2048 網頁遊戲，支援桌面鍵盤操作與手機觸控滑動。

## 遊玩方式

- **桌面**：使用方向鍵（↑↓←→）移動方塊
- **手機**：滑動螢幕移動方塊
- 相同數字的方塊碰撞時會合併，目標是湊出 2048！

## 功能

- ✅ 響應式設計，手機平板皆可遊玩
- ✅ 觸控滑動支援
- ✅ 分數計算與最高分記錄（localStorage）
- ✅ 動畫效果
- ✅ 遊戲結束偵測
